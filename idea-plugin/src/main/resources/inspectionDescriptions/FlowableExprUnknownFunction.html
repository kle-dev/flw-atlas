Reports function calls in Flowable expressions whose namespace or function name the Flowable expression catalog does not know.

Example: ${date:noww()} is reported with the suggestion did you mean 'now'?

Functions and namespaces the project registers itself (which the built-in catalog cannot know) can be added to the project's expression allowlist via Alt+Enter → Add … to Flowable expression allowlist — entries are stored in .idea/flowable-atlas.xml and shared with the team.

Applies to ${…} / #{…} backend fragments and {{…}} frontend fragments in BPMN/CMMN/DMN models, forms and pages, and to the Expression Playground.
